Demander l'obtention d'un compte authentifié

Les comptes authentifiés peuvent directement publier ou modifier des recettes, créer des évenements, etc, sans modération préalable. Le traitement des demandes se fait manuellement, et peut donc prendre un certain temps.

Vous devrez disposer d'un compte github avec la même adresse e-mail

Compte non authentifié

Il est possible de proposer des modifications au site en quelques clic. Pour cela, il suffit d'avoir un d'un compte github (plateforme sur lequel est hebergé le site). Ensuite, la premiere fois que vous essayerez de vous connecter à enka-cookbook, vos identifiants github vous serons demandés, et si vous n'avez pas obtenu de compte authentifié sur enka-cookbook, il vous sera demander d'accepter de faire un "fork" du dépot. Vous serez ensuite redirigé vers la page d'édition du site.

Les modifications que vous proposerez devront être valider par le compte administrateur du site avant d'être publié sur le site.
